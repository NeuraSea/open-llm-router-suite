# Credits

Open LLM Router Suite combines several open-source projects into a deployable
enterprise LLM gateway pattern.

- [New API](https://github.com/QuantumNous/new-api): primary API gateway,
  token, channel, usage, billing, and admin console foundation. Licensed under
  AGPLv3.
- [LibreChat](https://github.com/danny-avila/LibreChat): self-hosted chat UI
  used as the console chat entrypoint. Licensed under MIT.
- [CC Switch](https://github.com/farion1231/cc-switch): local Claude Code and
  Codex launcher/model switcher used by developer workstations. Licensed under
  MIT.
- [Casdoor](https://github.com/casdoor/casdoor): recommended OIDC/SSO identity
  provider that can front Feishu OAuth, WeChat QR login, and enterprise SSO.
  Licensed under Apache-2.0.

This repository's Router code, deployment glue, and documentation are released
under AGPLv3 to align with New API's license.
